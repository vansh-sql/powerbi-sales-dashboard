# Power BI Sales Dashboard (2022–2023)

A sales performance dashboard built in Power BI 
using real-world style computer hardware sales data.

## What's inside
- Supervisor-wise image slicer
- KPI Cards — Total Sales, Cost, Profit, Transactions
- Bar Chart — Quantity by Category
- Map Visual — Sales by Indian State
- Pie Chart — Brand-wise Sales split

## DAX Used
- SUMX for row-level Sales and Cost calculation

## Tools
Power BI Desktop | DAX | Data Modeling
